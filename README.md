# DoveCli Homepage

KIM SANG WOO가 운영하는 개인 개발자 계정 DoveCli의 공식 소개 페이지입니다.

## Scripts

- `npm run dev`: 로컬 개발 서버
- `npm run build`: Astro 타입 체크 및 정적 빌드
- `npm run preview`: 빌드 결과 미리보기

## Deployment

Cloudflare Pages 프로젝트명은 `dove-cli-homepage`입니다.

- Build command: `npm run build`
- Build output directory: `dist`
- Production branch: `main`
